.. _introduction:

Introduction
============

This extension provides a Data Consumer (see extension “tesseract” for a general explanation)
which relies on the Fluid template engine for rendering data provided by Tesseract Data Providers.
This makes it a very flexible tool.


.. _introduction-credits:

Credits
-------

This extension was originally developed by Fabien Udriot (Ecodev).

It takes advantage of all the power of Fluid, so credits also go to all
those who contributed to developing and enhancing Fluid.
